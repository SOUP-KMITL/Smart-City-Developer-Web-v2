CONNECTOR

Connect to your services with us for better monitoring.

ในปัจจุบันมีระบบเมืองอัจฉริยะอยู่หลายแบบ ตัวอย่างเช่น ระบบรถแท๊กซี่อัจฉริยะ ระบบตรวจวัดอากาศ ระบบตรวจคุณภาพน้ำ เป็นต้น หลายๆ ระบบที่มีอยู่มักจะมีระบบเป็นของตัวเอง ทางโครงการคลาวด์แพลตฟอร์มเพื่อการบูรณาการระบบเมืองอัจฉริยะต้องการรวบรวมข้อมูลจากแหล่งต่างๆ เพื่อให้ทางนักพัฒนาและผู้วิจัยสามารถทำไปใช้สร้างนวัตกรรมหรือแอพพลิเคชันเพื่อให้เกิดประโยชน์แก่ประชาชนทั่วไปได้ จึงได้ทำการพัฒนาโปรแกรมเชื่อมต่อ (connectors) สำหรับดึงข้อมูลดิบ (raw data) จากแหล่งข้อมูลต่างๆ แล้วนำเข้าฐานข้อมูลแบบกระจายตัว (distributed storage) ของระบบเมืองอัจฉริยะ

เนื่องจากบางระบบเกิดขึ้นมาหลายปี ตัวอย่างเช่น ข้อมูลเกี่ยวกับสังคมและสวัสดิการ ที่เก็บทางภาครัฐ ทางเว็บไซต์ https://data.go.th ซึ่งระบบเหล่านี้ไม่ได้ถูกสร้างเพื่อรองรับการเชื่อมต่อกับระบบอื่นๆ ได้อย่างอัติโนมัติ แต่ถ้าเป็นระบบที่เพิ่งถูกพัฒนาในช่วงไม่กี่ปีที่ผ่านมา ส่วนใหญ่ จะมี API ที่รองรับการเชื่อมต่อทำให้ สามารถเชื่อมต่อกับระบบคลาวด์แพลตฟอร์มเพื่อบูรณาการเมืองอัจฉริยะได้ทันที เพื่อรองรับการนำเข้าข้อมูลและเชื่อมต่อกับทั้งระบบแบบเก่าและแบบใหม่ เราจึงได้พัฒนาโปรแกรมเชื่อมต่อขึ้นมา และรวบรวมแหล่งข้อมูลต่าง ๆ ให้มากที่สุด เพื่อให้ง่ายต่อการเข้าถึง และสามารถนำข้อมูลไปใช้เพื่อพัฒนาระบบเมืองอัจฉริยะ

Type of Connector

1. แหล่งข้อมูลที่มี API ในการดึง ผู้ใช้สามารถนาข้อมูลเข้าโดยการเชื่อม API จากระบบของตน เข้าในระบบคลาวด์แพลตฟอร์มเพื่อการบูรณาการเมืองอัจฉริยะได้ทันที

2. แหล่งข้อมูลไม่มี API รองรับการดึงข้อมูลออก

i. โปรแกรมเชื่อมต่อที่รองรับข้อมูลแบบไฟล์ (File-based Connector)

๐ กรณีที่ไฟล์อยู่บนเว็บไซต์ (Website) ตัวอย่างเช่น data.go.th และ dbd.go.th กรณีนี โปรแกรมเชื่อมต่อจะทาการดึงข้อมูลผ่านส่วนประกอบต่าง ๆ ใน HTML เพื่อทาการดาวน์โหลดไฟล์เหล่านั นเข้าสู่ระบบ

๐ กรณีที่ไฟล์อยู่บนเซิร์ฟเวอร์ (Server) ต้องใช้ SCP

ii. โปรแกรมเชื่อมต่อที่รองรับข้อมูลแบบการร้องขอผ่านเว็บ (Web-request-based Connector)
